# JobTracker

🚧 A personal project for tracking new job openings across company career pages.

The goal of this project is to periodically scrape selected company websites, identify newly posted roles, and eventually send notifications when relevant openings appear.

## Features

* Scrape job postings from company career pages
* Detect newly added openings
* Store historical results locally
* Support multiple companies

## Future Improvements

* Email or Telegram notifications
* Basic filtering by keywords/location
* Simple dashboard or web interface
* Scheduled automated runs

## Tech Stack

* Python
* BeautifulSoup / Requests
* Pandas
* GitHub Actions (planned)

## Status

Currently in early MVP stage.
